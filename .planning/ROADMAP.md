# Roadmap: Promptys

## Overview

Promptys é construído seguindo a progressão natural do usuário: primeiro consome (L1), depois valida (L2), depois cria (L3). As fases do roadmap espelham essa progressão. A v1.0 (MVP) entregou a experiência completa L1 → L2 → L3 com gap closure de integração.
